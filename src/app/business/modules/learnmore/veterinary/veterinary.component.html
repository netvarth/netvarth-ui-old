Public Search
Public Search
  1. A doctor is required to click on the “TURN ON“ button to enable the Public Search if he/she wants to be visible in the customer's search results.
  2. If a doctor has disabled the Public Search, customers cannot see the doctor’s profile in their search results.
  3. customers can search for a doctor with any attributes that are listed under the Profile and Search section. These attributes can include the business name, Jaldee verified levels,Jaldee Search Keyword, Basic Information, Location, Location Amenities, Specialization, Languages Known, Additional Info.
  4. Even if a doctor turns Public Search to off, customers can check-in with the same doctor from their “Favourites“ (if already added doctor to Favourites List) list or using custom url.
  5. During the initial profile setup, a doctor can proceed with setting the Profile Name & Summary, Location, Queues, Service and Public Search status. The doctor can setup or update these informations here.
    • If the doctor logs out of the website after setting the password, the account will not be active. To activate the account, the doctor has to completely fill basic details and turn on the Public Search. At this point, a default service gets created which can be edited in Services under the Check-In Manager.
    • If the doctor logs out of the website after setting up the location, a default service time window gets created. The doctor can edit this from here.

    Calculation of waiting time
    Waiting time simply refers to the time a customers has to wait before he/she can avail the services offered at the premise.
    • Automatic (Machine Learning) : Using a machine learning algorithm, the system automatically calculates the turnaround time for each service by analyzing a doctor’s service history.
    • Manual : Using this option, a doctor can fix a wait time which will be applied to all the check-ins (for example, If a doctor sets 10 minutes of manual time, by default all the customers will get 10 minutes of waiting time.) Manual time should be used when a doctor can guarantee to serve all the customers in the given time. To do so, it is advisable that the service doctor is equipped with enough resources. An additional option is available for doctors choosing “Manual�? wait time calculation. He/She can edit individual check-ins wait time from the check-in details in doctor dashboard.
    • Token System(Do not show waiting time for customers): If a service doctor chooses this option, then all the customers will receive and see a token number instead of any waiting time.
    • No Token : If a service doctor chooses this option, the customers won’t be able to see the waiting time as well as the token number. Here a customer can see the number of persons ahead of him/her.
