Public Search
  1. A provider is required to click on the “TURN ON“ button to enable the Public Search if he/she wants to be visible in the client's search results.
  2. If a provider has disabled the Public Search, clients cannot see the provider’s profile in their search results.
  3. clients can search for a provider with any attributes that are listed under the Profile and Search section. These attributes can include the business name, Jaldee verified levels,Jaldee Search Keyword, Basic Information, Location, Location Amenities, Specialization, Languages Known, Additional Info.
  4. Even if a provider turns Public Search to off, clients can check-in with the same provider from their “Favourites“(if already added provider to Favourites List) list or using custom url.
  5. During the initial profile setup, a provider can proceed with setting the Profile Name & Summary, Location, Queues, Service and Public Search status. The provider can setup or update these informations here.
    • If the provider logs out of the website after setting the password, the account will not be active. To activate the account, the provider has to completely fill basic details and turn on the Public Search. At this point, a default service gets created which can be edited in Services under the Check-In Manager.
    • If the provider logs out of the website after setting up the location, a default service time window gets created. The provider can edit this from here.

    Calculation of waiting time
    Waiting time simply refers to the time a clients has to wait before he/she can avail the services offered at the premise.
    • Automatic (Machine Learning) : Using a machine learning algorithm, the system automatically calculates the turnaround time for each service by analyzing a provider’s service history.
    • Manual : Using this option, a provider can fix a wait time which will be applied to all the check-ins (for example, If a provider sets 10 minutes of manual time, by default all the clients will get 10 minutes of waiting time.) Manual time should be used when a provider can guarantee to serve all the clients in the given time. To do so, it is advisable that the service provider is equipped with enough resources. An additional option is available for providers choosing “Manual“ wait time calculation. He/She can edit individual check-ins wait time from the check-in details in provider dashboard.
    • Token System(Do not show waiting time for clients): If a service provider chooses this option, then all the clients will receive and see a token number instead of any waiting time.
    • No Token : If a service provider chooses this option, the clients won’t be able to see the waiting time as well as the token number. Here a client can see the number of persons ahead of him/her.
