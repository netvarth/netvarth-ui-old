Public Search
  1. A doctor is required to click on the “TURN ON" button to enable the Public Search if he/she wants to be visible in the patient’s search results.
  2. If a doctor has disabled the Public Search, patients cannot see the doctor’s profile in their search results.
  3. patients can search for a doctor with any attributes that are listed under the Profile and Search section. These attributes can include the business name, Jaldee verified levels, AdWord, Basic Information, Location, Location Amenities, Specialization, Languages Known, Additional Info.
  4. Even if a doctor turns Public Search to off, patients can check-in with the same doctor from their "Favourites"(if already added doctor to Favourites List) list or using custom url.
  5. During the initial profile setup, a doctor can proceed with setting the Profile Name & Summary, Location, Queues, Service and Public Search status. The doctor can setup or update these informations here.
    • If the doctor logs out of the website after setting the password, the account will not be active. To activate the account, the doctor has to completely fill basic details and turn on the Public Search. At this point, a default service gets created which can be edited in Services under the QManager.
    • If the doctor logs out of the website after setting up the location, a default service time window gets created. The doctor can edit this from here.

    Privacy Settings
    Getting to know and controlling your privacy settings (Phone number & Email address) is an important strategy to protect your information. Here the system allows the doctor to add, edit or delete both phone number and email address.
    Phone
    1. A doctor can view the phone number used to create a Jaldee account. By default, the visibility of the number is set to none.
    2. The three dots menu on the right corner has two options, namely, Edit and Delete.
    3. If a doctor wants to edit an entry, he/she can click on the “Edit" button. Here, the doctor can change the phone number, phone label (for example, mobile, emergency, mainline or others) and also the visibility of the number.
    4. The phone number is shown in the profile’s detailed view in patient's dashboard. This helps the patients to contact doctor easily.
    5. The doctor has an option to restrict the visibility of the phone number. Two options to filter the visibility are:
      • Public(All can see) : When set to Public, everyone can see the doctor’s number.
      • Private(None can see) : When set to Private, only the doctor can view the phone number.
    6. A doctor can also delete a phone number. To do so, click “Delete" from the three dots menu on the right corner.
         Email
    1. A doctor can add, edit and delete the email address here.
    2. The three dots menu on the right corner has two options, viz. Edit and Delete.
    3. In case, a doctor wants to edit an entry, he/she can click on the “Edit" button. Here, the doctor can change the email address, email label (for example,Office Email Id, Personal Email Id, etc.) and also the visibility of the email address.
    4. The email address provided here is shown to the patients in the detailed view of the doctor profile.
    5. The doctor has an option to restrict the visibility of the email address. Three options to filter the visibility are:
      • Public(All can see) : When set to Public, everyone can see the doctor’s email.
      • Private(None can see) : When set to Private, only the doctor can view the email.
    6. A doctor can delete an email address. To do so, click “Delete" from the three dots menu on the right corner.
    7. Jaldee.com uses the email address to communicate the general notifications to the doctor, such as notifications on invoice, Jaldee coupons, discounts, etc.
